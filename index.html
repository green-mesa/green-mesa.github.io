Hyperbone

Hyperbone is a non-framework framework, built on Backbone.js, for building client side applications consuming JSON HAL.

By 'Non-Framework', we mean that each component is distributed via the Component package manager. Use one or all of the components as you see fit.

Activity on this project is high at the moment. We hope to get a fully annotated demo application up and running soon.

Component is a great package manager, but the build tool adds a significant file size overhead when dealing with a large number of Components. Hyperbone Build allows you to use Component as your package manager and compile your source as a self-bootstrapping application with minimal module related detritus.
