HalogenJS

HalogenJS is a HAL/HAL-O JSON framework for working natively with hypermedia resources in the browser.

Animation and Experimental Modules

Many additional modules are available on Charlotte Gore's personal Github here, which tend to be available on npm with the gm- prefix.

The modules are CommonJS with fluent interfaces, and many are intended for use for animation, games and web audio.

Dom / Value / Traverse / CSS

Green Mesa maintain forks of component-dom and some of its dependencies which fixes issues around dealing with forms. This is a dependency for HalogenJS as we do not want to have jQuery as a dependency.

Route

Green Mesa maintain a fork of component-route in order to have it available as an npm package.

Hyperbone

Hyperbone is the a collection of components for a hypermedia framework that has since become HalogenJS. These repositories are maintained so that they will continue to be available through Component, and they're also maintained as NPM packages.

Backbone

Backbone refactored into individual components. A dependency for HalogenJS (and Hyperbone).

We have:

  • Backbone Events
  • Backbone Model
  • Backbone Collection
